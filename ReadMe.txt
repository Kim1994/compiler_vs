compiler_vs
本控制台程序实现了正则表达式解析器，可配置语法分析器（ll1）

使用方法：
1、正则分析器

a、将ll1.cpp文件中的main函数注释掉，运行compiler_vs.cpp中的main函数。

b、输入一条正则表达式（支持符号“|”和“（”“）”和“*”）

c、等待分析完毕后  再输入一段字符串，即可显示该字符串是否与之前的正则表达式相匹配

2、可配置语法分析器

a、将compiler_vs.cpp文件中的main函数注释掉，运行ll1.cpp中的main函数。

b、输入一串符合ll(1)的语句，最后一行为end，例（加减乘运算）：
~e->~t ~E
~E->+ ~t ~E
~E-> 
~t->~f ~T
~T->* ~f ~T
~T-> 
~f->( ~e )
~f->id
end
若为空则输入空格，如第三行。

c、输入一串token，每个token用空格隔开，最后加上$,即可显示是否匹配，例：
id * ( id + id ) $
